# Quote API

### Requirements
```
node >= 12
```

### Getting started

```
# Clone this repo:
git clone https://github.com/nexusmutual/quote-api.git

# Install dependencies
npm install

# Copy .env.sample to .env and edit accordingly
cp .env.sample .env
$EDITOR .env

# Run it
npm start
```
